# Anniversary Map

Static deploy package for GitHub Pages.

## Deploy To GitHub Pages

1. Create a new GitHub repository.
2. Upload everything in this folder to the repository root, including `index.html`, `.nojekyll`, and the assets folder.
3. In GitHub, open `Settings > Pages`.
4. Set `Source` to `Deploy from a branch`.
5. Select the branch you uploaded to, usually `main`, and choose `/ (root)`.
6. Save. GitHub Pages will publish the site after the Pages build finishes.

The entry file is `index.html`.
